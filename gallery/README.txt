GALLERY IMAGES FOLDER
=====================

Place your venue images here. Recommended:

File naming convention:
- venue-1.jpg, venue-2.jpg, venue-3.jpg, etc.
- event-1.jpg, event-2.jpg, etc.
- decoration-1.jpg, decoration-2.jpg, etc.

Recommended image specs:
- Format: JPG or WebP (WebP preferred for smaller size)
- Resolution: 1200x800 pixels (landscape)
- File size: Under 500KB each (compress using tinypng.com or squoosh.app)

Current placeholder images expected:
1. venue-main.jpg      - Main hall exterior or interior
2. venue-large.jpg     - 1000+ capacity hall
3. venue-small.jpg     - 500+ capacity hall
4. dining-area.jpg     - Dining shed/area
5. decoration-1.jpg    - Decoration sample 1
6. decoration-2.jpg    - Decoration sample 2
7. bride-room.jpg      - Bride preparation room
8. event-1.jpg         - Past event photo 1
9. event-2.jpg         - Past event photo 2
10. event-3.jpg        - Past event photo 3

To update images:
Simply replace the files with same names. The website will automatically use the new images.
